# clipso — context

## stack
- bash 5.x, Termux + Debian + macOS
- backends: termux-clipboard-set | wl-copy | xclip | OSC52 | pbcopy-forward
- install.sh: idempotent, ln -sf, chmod +x; enlaza $PREFIX/bin Y ~/.local/bin (stale guard)

## architecture
- privacy_check(): single awk pass, cero forks por línea
  - CRED: word-boundary + assignment context + value ≥5 chars + non-placeholder
  - PRIV-IP: RFC1918 strict + valid_ip() octet validation
  - MAC: strict 6-octet; excluye 00:00:00:00:00:00 y ff:ff:ff:ff:ff:ff
  - PUB-IP: valid_ip() + not private + not safe-DNS + no version-like context
  - Self-flagging del propio source eliminado
- display_with_privacy(): imprime líneas numeradas a /dev/tty; flagged en rojo inline
  - Globals: PRIVACY_HITS (int), PRIVACY_INFO_FILE (linenum TAB tag TAB masked)
  - Clipboard: flagged lines → "# censored line" (preserva line count)
  - Warn AFTER output; BYTES recomputed si hits>0
- _fmt_size(): pure bash, sin bc (Termux compat)
- CLIP_FORWARD_USED: flag para pbcopy-forward en ok() line
